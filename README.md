## Beyond Linear Decision Rules: LLM-Guided Representation Discovery for Data-Driven Optimization

**Huan Zhang**, **Yang Wang**, **Hanzhang Qin**, and **Yue Zhao**

- Huan Zhang and Yang Wang: School of Management, Northwestern Polytechnical University
- Hanzhang Qin: Department of Industrial Systems Engineering and Management, National University of Singapore
- Yue Zhao: HSBC Business School, Peking University

[[SSRN Paper](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=7127438)]

## Overview

Stochastic optimization often restricts recourse decisions to affine parameterizations for computational tractability, potentially creating a substantial and irreducible approximation gap. Large language models (LLMs) offer new opportunities to discover richer problem-specific structures that can help narrow this gap. We propose *Generative Stochastic Optimization* (GenSO), a framework that uses LLMs to discover representations for adaptive recourse while preserving the rigor and tractability of optimization models.

The key component of GenSO is the *Generative Linear Decision Rule* (GenLDR), in which recourse decisions are expressed as linear combinations of closed-form nonlinear basis functions generated by LLMs from problem structure and domain knowledge. Consequently, GenSO increases the expressiveness of recourse decisions without compromising tractability and interpretability.

We establish finite-sample guarantees for fixed generated representations and further derive a performance bound under an assumption motivated by empirical scaling laws of LLMs. We evaluate GenSO on a multi-period newsvendor problem and a data center location problem with multi-period job scheduling calibrated using real-world datasets. Across both settings, GenSO consistently outperforms existing benchmarks in out-of-sample evaluations while uncovering interpretable nonlinear decision structures that are difficult to identify through conventional manual design.

## Projects in This Repository

This repository contains three independent projects:

- [`GenSO/`](GenSO/): the GenSO implementation and LLM-guided representation-discovery experiments.
- [`DataCenter-demo/`](DataCenter-demo/): standalone data center location and multi-period job-scheduling experiments.
- [`MNV-demo/`](MNV-demo/): standalone multi-period newsvendor experiments.

## Highlights

- **LLM-guided representation discovery:** GenSO uses problem structure and domain knowledge to generate nonlinear basis functions for adaptive recourse.
- **Tractable and interpretable optimization:** GenLDR retains a linear optimization layer while enriching the underlying decision representation.
- **Theoretical guarantees:** The framework provides finite-sample guarantees for fixed generated representations and a scaling-law-motivated performance bound.
- **Realistic evaluations:** Experiments cover a multi-period newsvendor problem and a data center location and job-scheduling problem calibrated with real-world data.

## Getting Started

From the repository root:

```bash
cd GenSO
python -m pip install -e ./eoh
```

Set your DeepSeek API key through the `DEEPSEEK_API_KEY` environment variable. For example, in PowerShell:

```powershell
$env:DEEPSEEK_API_KEY = "your-api-key"
python eoh/src/eoh/test/run.py
```

The optimization experiments also require the solver and Python packages used by the selected problem instance, including Gurobi for the supplied mathematical programming models.
